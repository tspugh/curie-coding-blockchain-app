Adalimumab — Specialty (Tier 5)

Status: Covered with restrictions — prior authorization, step therapy, quantity limit.

Clause PD-ADA-01 — Prior authorization criteria

  • Indication: moderate-to-severe chronic plaque psoriasis.
  • Disease severity: BSA ≥ 10%, or PASI ≥ 12.
  • Trial-and-failure or intolerance of ≥ 1 preferred conventional systemic therapy (e.g. methotrexate).
  • No active serious infection; latent TB screening performed.

Public standard: consistent with the openFDA HUMIRA label indication for moderate-to-severe chronic plaque psoriasis. Compliant.

Clause PD-ADA-02 — Step-therapy exception

An exception applies when a preferred conventional systemic agent has been tried and failed, or is contraindicated/not tolerated, with documentation.

Benchmark cap

Covered amount = min(requested, 4200) (unitless marker; see price-benchmarks.md). The agent never sets the amount (R6a).

Ruling guidance

Return approve when PD-ADA-01 criteria and a valid PD-ADA-02 step-therapy exception are documented; need_more_evidence when documentation is incomplete; deny when a preferred agent has not been tried and no exception applies. If a relied-on clause contradicts the cited public standard, flag it (policy_invalid) rather than applying it.
